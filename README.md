# FINAL
CVSU-Verification-System

CVSU Carmona Alumni Profile with Blockchain Document Verification System

## Deployed Services

The system is deployed on Render with the following services:

- Backend API: [https://final-ecri.onrender.com](https://final-ecri.onrender.com)
- Frontend: [https://alumni-frontend-4r7o.onrender.com](https://alumni-frontend-4r7o.onrender.com)

## Project Structure

- `frontend/`: React frontend application with Vite
- `backend/`: FastAPI backend with MongoDB and Hyperledger Fabric integration
- `wallet/`: Blockchain wallet files and certificates
- `scripts/`: Utility scripts for setup and maintenance
